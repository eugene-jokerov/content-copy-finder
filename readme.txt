Check the text for uniqueness and plagiarism search
 
== Description ==

This plugin allows you to check the texts for uniqueness and find plagiarism on other sites( for Russian Language ). 

Плагин позволяет проверить уникальность текстов и найти плагиат на других сайтах. Для проверки уникальности и поиска плагината используется API сервиса content-watch.

В основе сервиса лежит эффективный многоступенчатый алгоритм, который позволяет быстрее и точнее проверять уникальность.

Чтобы использовать возможности плагина, вы должны получить ключ API на странице [https://content-watch.ru/api/](https://content-watch.ru/api/).

== Installation ==

= From within WordPress =
1. Visit 'Plugins > Add New'
1. Search for 'Content Copy Finder'
1. Activate Content Copy Finder from your Plugins page.
1. Go to Settings -> Content Copy Finder and customize behaviour as needed

= Manually =
1. Unzip the archive on your computer  
2. Upload the `content-copy-finder` folder to the `/wp-content/plugins/` directory
3. Activate the Content Copy Finder plugin through the 'Plugins' menu in WordPress
4. Go to Settings -> Content Copy Finder and customize behaviour as needed

= После установки =
1. Перейдите по ссылке и получите ключ API [content-watch API key](https://content-watch.ru/api/)
2. Введите ключ API на странице настроек плагина
3. Теперь Вам доступен интерфейс проверки уникальности

== Frequently Asked Questions ==
